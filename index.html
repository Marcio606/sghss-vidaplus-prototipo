SGHSS - Sistema de Gestao Hospitalar e de Servicos de Saude

VidaPlus - Instituicao de Saude

Desenvolvido por: Marcio Machado Moreira - RU: 4543545

Funcionalidades Implementadas

  • Cadastro e gestao de pacientes
  • Gestao de profissionais de saude
  • Agendamento de consultas
  • Prontuarios eletronicos
  • Telemedicina com videochamadas
  • Prescricoes digitais
  • Gestao de leitos hospitalares
  • Relatorios administrativos
  • Controle de acesso e seguranca
  • Conformidade com LGPD

Tecnologias Utilizadas

  • Java 11
  • Spring Boot 2.7.14
  • MySQL 8.0
  • Maven
  • Spring Security + JWT
  • OpenAPI 3 (Swagger)
  • Docker

Como Executar

  1. Clone o repositorio
  2. Execute: docker-compose up -d
  3. Acesse: http://localhost:8080/sghss/swagger-ui.html
  4. Teste a API via Swagger UI
